查找元素

  • var myElement = $("#id01"); -- id （js中，var myElement = document.getElementById("id01");）
  • var myElements = $(".intro"); -- 类名 （js中，var myElements = document.getElementsByClassName("intro");）
  • var myElements = $("p"); -- 类名 （js中，var myElements = document.getElementsByTagName("p");）
  • var myElements = $("p.intro"); -- CSS选择器 （返回包含 class="intro" 的所有p元素的列表，js中，var myElements = document.querySelectorAll("p.intro");）
