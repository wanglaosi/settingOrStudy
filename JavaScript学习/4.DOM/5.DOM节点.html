• parentNode -- 父节点
  • childNodes[n] -- 第n个子节点
  • firstChild -- 第1个子节点
  • lastChild -- 最后1个字节点
  • nextSibling -- 下1个兄弟节点 （nextElementSibling忽略文本，解决换行会导致获取到#text的问题）
  • previousSibling -- 上1个兄弟节点
  • nodeName -- 节点名
  • nodeValue -- 节点值
  • nodeType -- 节点类型
  • 创建新的 HTML 元素 (节点，添加到尾部) - appendChild()
  • 创建新的 HTML 元素 (节点，添加到头部) - insertBefore()
  • 移除已存在的元素 - 要删除元素的父元素.removeChild(要删除的元素);
  • 替换 HTML 元素 - replaceChild()
