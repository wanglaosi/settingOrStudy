使用 document 对象来访问和操作 HTML 的实例。

查找 HTML 元素：

  • document.getElementById(id) -- id
  • document.getElementsByClassName(name) -- 类名
  • document.getElementsByTagName(name) -- 标签名

改变 HTML 元素：

  • 获取的元素.innerHTML = 新的显示内容 -- 改变元素的 inner HTML
  • 获取的元素.属性 = 属性值 -- 改变 HTML 元素的属性值
  • 获取的元素.setAttribute(属性, 属性值) -- 改变 HTML 元素的属性值
  • 获取的元素.style.属性 = 属性值 -- 改变 HTML 元素的样式

添加和删除元素：

  • document.createElement(获取的元素) -- 创建 HTML 元素
  • document.removeChild(获取的元素) -- 删除 HTML 元素
  • document.appendChild(获取的元素) -- 添加 HTML 元素
  • document.replaceChild(获取的元素) -- 替换 HTML 元素
  • document.write(text) -- 写入 HTML 输出流

添加事件处理程序：

  • document.getElementById(id).onclick = function(){ ... } -- 向 onclick 事件添加事件处理程序
